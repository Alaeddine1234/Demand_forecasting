# Demand Forecasting App

A machine learning web application that predicts product demand based on key business factors.

## Demo
Enter product details (price, discount, promotion, inventory level, competitor pricing, category) and get an instant demand prediction.

## Technologies
- Python
- XGBoost
- Scikit-learn
- Pandas
- Streamlit

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
